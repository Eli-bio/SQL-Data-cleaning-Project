This project focuses on cleaning and preparing raw data using SQL. The dataset contains real-world messy data that requires preprocessing before analysis.



🎯 Objectives
Handle missing values

Remove duplicates

Standardize inconsistent data

Prepare clean dataset for analysis



🛠️ Tools & Technologies
SQL (MySQL)

Data Cleaning Techniques

The dataset used in this project is related to layoffs and contains information about companies, locations, industries, and layoffs.



🔧 Data Cleaning Steps

Removed duplicate records

Handled null and blank values

Standardized text fields (e.g., company names, industries)

Converted date formats

Removed unnecessary columns

Practical SQL data cleaning techniques

Importance of data preprocessing in analytics

Writing efficient and readable SQL queries
